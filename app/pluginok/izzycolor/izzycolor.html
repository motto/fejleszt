Color Picker - Sample

How to Click here to download izzyColor plugin

First, define SCRIPT tags in HEAD. If you already have loaded jQuery, you can skip first line: <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js" type="text/javascript"></script>
<script src="izzyColor.js"></script> As soon as it's defined, all input fields with class "izzyColor" will be converted to Color Picker: <input type="text" value="#CCCC66" class="izzyColor" id="color1"> It's important, that each text input should have its own, unique ID defined.
